使い方

<script src="https:~~"></script>は必ず</body>の直前に入れてください。

<body>
  ...

    console.log('Hello!');
  </js-window>
  ...
  <script src="https://makeplayonline.onrender.com/Static/API/JS-Window/1.01.01/JS-Window.js"></script>
</body>
