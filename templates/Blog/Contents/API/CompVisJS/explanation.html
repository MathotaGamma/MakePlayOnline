メソッドの詳細

  • 引数ありのメソッド

    • C.round(n)

      Cの実部、虚部をそれぞれ特定の桁数で四捨五入する。
      n>0の場合、小数第n位で四捨五入し、
      n=0の場合、そのまま四捨五入して整数値にして、
      n<0の場合、10nの位で四捨五入する。

    • C.add(c)

      Cにcを足す。
      例:
      const C = new CompVis(1,2); const c = new CompVis(-2,1); console.log(C.add(c).str);
